# >>> Shoutbox <<<
A basic public group chat (proof of concept) using Socket.io.

### Setup instructions
- `npm install`
- change the mongodb connection url in `server/mongoose.js` file (line no 6)
- change the port number in `app.js` file (line no 14 `_PORT`) (OPTIONAL)
